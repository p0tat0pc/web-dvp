Projects

Weather App

The Weather App is a responsive web application that allows users to check the current weather conditions for any city around the world. Users can enter the name of a city, and the app will display the current temperature, humidity, wind speed, and a brief description of the weather (e.g., cloudy, sunny). The app fetches real-time weather data from the OpenWeatherMap API and updates the UI accordingly.

ToDo List App

The ToDo List App is a simple yet effective task management application that allows users to keep track of their daily tasks. Users can add, edit, and delete tasks as needed. Each task can be marked as complete or incomplete, providing a visual indication of progress. The app stores the tasks in the browser's local storage, ensuring that the data persists even after the browser is closed.
